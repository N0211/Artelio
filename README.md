# Artelio – AI-Powered Artist Portfolio Platform

## Overview
Artelio is a Django-based web application that integrates AI to help users discover, compare, and collaborate with artists.

## Features
- User authentication
- Artist portfolios
- Artwork uploads
- AI candidate search
- Artist comparison
- Collaboration suggestions
- Style analysis

## Technologies Used
- Python / Django
- MySQL
- OpenAI API (GPT-4o-mini)
- Hugging Face CLIP
- HTML / CSS

## Setup Instructions
1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add `.env` file with OpenAI API key
4. Run server:
   python manage.py runserver

## Notes
- AI fallback mechanism not fully implemented
- Project is a working prototype with core features complete
